설명: surround.vim : Delete/change/add parentheses/quotes/XML-tags/much more with ease

코딩할 때 텍스트를 태그, 따옴표, 괄호 등으로 감싸거나 없애는 경우가 많은데 이를 쉽게 해주는 플러그인이다.

플러그인 주소: http://www.vim.org/scripts/script.php?script_id=1697

GitHub 주소: https://github.com/tpope/vim-surround

1. 설치

cd ~/.vim/bundle
git clone git://github.com/tpope/vim-surround.git

2. 사용법

1) 쌍따옴표를 홑따옴표로 바꾸기

"Hello world!"

위와 같은 텍스트가 있을 때, 저 안 아무데나 커서를 두고 일반모드에서 cs"' 를 하면 이렇게 바뀐다.

'Hello world!'

2) 홑따옴표를 태그로 바꾸기

'Hello world!'

위와 같은 상태에서 cs'<q> 라고 입력하면 이렇게 바뀐다.

<q>Hello world!</q>

3) 태그를 쌍따옴표로 바꾸기

<q>Hello world!</q>

위와 같은 상태에서 cst" 라고 입력하면 이렇게 바뀐다.

"Hello world!"

4) 따옴표 지우기

 

"Hello world!"

위와 같은 상태에서 ds" 라고 입력하면 이렇게 바뀐다.

Hello world!

5) 커서가 있는 단어를 대괄호로 감싸기

Hello world!

위와 같은 상태에서 Hello 위에 커서를 둔다(블럭으로 지정하면 안된다). 그리고 나서 ysiw] 를 입력하면 다음과 같이 바뀐다. 여기서 iw는 텍스트 덩어리를 가리킨다.

[Hello] world!

6) 대괄호를 중괄호로 바꾸기

[Hello] world!

위와 같은 상태에서 Hello에 커서를 두고 cs]} 를 입력하면 다음과 같이 바뀐다.

{Hello} world!

만약 cs]} 대신 cs]{ 라고 입력하면 다음과 같이 공백이 추가된다.

{ Hello } world!

7) 한 줄 전체를 소괄호로 감싸기

{ Hello } world!

위와 같은 상태에서 yssb 또는  yss)를 입력하면 다음과 같이 바뀐다.

({ Hello } world!)

8) 괄호 다 지우기

({ Hello } world!)

이 상태에서 ds}ds)를 입력하면 다음과 같이 바뀐다.

Hello world!

9) 태그로 감싸기

Hello world!

위와 같은 상태에서 hello에 커서를 두고 ysiw<em>을 입력하면 다음과 같이 바뀐다.

<span>Hello</span> world!

끝.
