플레인 텍스트는 위대하다. LaTeX은 플레인 텍스트의 위대함을 제대로 보여주는 시스템이다.

PythonTeX은 파이썬과 LaTeX을 잘 섞어놓은 프로그램이다.

유툽(PythonTeX)에서 보고 감동받았는데 오늘에야 테스트를 해봤다.

이렇게 작성한 후,

\documentclass{article}
\usepackage{fontspec}
\usepackage{pythontex}
\begin{document}
hello
\begin{pyconsole}
print("Hello Python")
\end{pyconsole}
\end{document}

latex과 pythontex을 실행하면 다음과 같은 결과를 얻을 수 있다.

[caption id="attachment_384" align="alignnone" width="224"] 최종 파일인 pdf에서 캡쳐한 그림[/caption]

 

과정을 간단히 설명하면 이렇다.

  1. 텍 문서(*.tex)를 작성한다. 이 때 입력하고 싶은 파이썬 코드를 pythontex이 제공하는 명령이나 환경에 넣는다.
  2. 텍 문서를 컴파일(texify)한다.
  3. 텍 문서를 컴파일하면 텍 시스템은 여러 부수파일을 만든다. 우리는 pythontex을 사용했기 때문에 pytxcode라는 부수파일이 생긴다. 일반적인 texify 결과에는 없는 파일이다.
  4. pythontex2.py로 pytxcode파일을 처리한다.
  5. 다시 한 번 텍 문서를 컴파일한다.
  6. 짜잔~ 끝.

유투브에서 보면 알겠지만 pyconsole외에 다양한 기능을 제공한다. 산술연산은 물론 그래프로 표현하는 것도 가능하다.
