LaTeX 문서에 프로그래밍 코드를 넣을 일이 있을 때, 당연히 키워드 등이 하이라이팅 되는 게 보기 좋다.

\documentclass{article}
\usepackage{minted}
\begin{document}
\begin{minted}{python}
class MyClass(object):
	'''
	Sample Class
	'''
	def __init__(self, name):
		self.myname = name  # name is your name

\end{minted}
\end{document}

minted 패키지는 다 좋은 데 쪽나눔을 지원하지 않는다. 이 한가지 이유 때문에 나는 minted 패키지 대신 verbments 패키지를 많이 사용한다.

\documentclass{article}
\usepackage{verbments}
\begin{document}
\begin{pyglist}[language=python, encoding=utf8]
class MyClass(object):
	'''
	Sample Class
	'''
	def __init__(self, name):
		self.myname = name  # name is your name

\end{pyglist}
\end{document}

패키지 이름은 verbments지만, 환경은 pyglist이다. 기본 인코딩이 latin1이므로 utf-8로 지정하고 싶으면 encoding=utf8옵션을 주어야 한다.

이렇게 작성하고 다음과 같이 컴파일한다.

pdflatex -shell-escape latexfilename

xetex으로 하고 싶으면

xelatex -shell-escape latexfilename

결과:

만약 beamer에서 쓰고 싶다면, beamer를 사용할 때 옵션을 주어야 한다.

\begin{frame}[fragile]
	\begin{minted}{python}
	class MyClass(object):
	    '''
	    Sample Class
	    '''
	    def __init__(self, name):
	        self.myname = name  # name is your name

	\end{minted}
\end{frame}

와 같이 [fragile]옵션을 주어야 에러없이 컴파일이 된다.
