PyQt로 메뉴가 있는 윈도우를 만들 때, 하위 메뉴를 만드는 방법에 대한 간단한 예.

class MyWin(QMainWindow):
	def __init__(self, parent=None):
		super(MyWin, self).__init__(parent)
		self.Actions()
		self.Menu()

def Actions(self):
	self.settingAction = QAction(QIcon(':/setting.png'), '&Setting', self)
	self.openDirAAAAction = QAction(QIcon(':/folder.png'), '&AAA', self)
	self.openDirBBBAction = QAction(QIcon(':/folder.png'), '&BBB', self)	
	self.openDirCCCAction = QAction(QIcon(':/folder.png'), '&CCC', self)

def Menu(self):
	menubar = self.menuBar()

	toolsMenu = menubar.addMenu("&Tools")
	toolsMenu.addAction(self.settingAction)
	openDirMenu = toolsMenu.addMenu(QIcon(':/folder.png'), "&Open Dir")
	openDirMenu.addActions((self.openDirAAAAction,self.openDirBBBAction,self.openDirCCCAction))

 

  1. menuBar()로 메뉴객체를 만든다 -> menubar
  2. 이 메뉴객체에 메뉴를 추가한다 -> toolsmenu
  3. 추가한 메뉴에 액션을 추가한다. 이 때 toolsmenu에 직접 추가하면 [Tools]메뉴에  바로 보인다-> Setting
  4. toolsmenu에 addMenu메소드로 하위메뉴를 추가한다 -> &Open Dir
  5. 추가한 하위메뉴(&Open Dir)에 액션을 추가한다 -> openDirAAAAction, openDirBBBAction, openDirCCCAction
  6. 5번에서 추가한 액션들은 [Tools]-[Open Dir]메뉴 아래에 나타난다.
