jQuery에서 키보드 이벤트를 잡을 때는 keypress, keydown, keyup 과 같은 키워드를 이용한다.

하지만 이 키워드를 그냥 이용하기 보다는 엔터키인지 확인하는 경우가 많다.

사용자가 엔터키를 눌렀는지 확인하는 방법.

$('#my-id').live('keypress', function(e) {
	if (e.which == 13) {/* 13 == enter key@ascii */
		alert("you pressed enter key");
	}
}

e.which가 핵심. which를 써야 된다. keycode가 아니다!
