  1. virtualenv 설치
  2. pythonz 설치(https://github.com/saghul/pythonz) 
    $ pythonz install 3.3.3
  3. virtualenv로 가상환경 만들기 
    $ virtualenv --python=.pythonz/pythons/CPython-3.3.3/bin/python --no-site-packages mypython3
  4. $ source mypython3/bin/activate
  5. (mypython3)$

3번에서 가상환경 만들 때 주의할 점이 있다. pythonz의 경로를 이런 식으로 지정하면 안된다.

$ virtualenv --python=~/.pythonz/pythons/CPython-3.3.3/bin/python --no-site-packages mypython3

만약 위와 같이 경로를 지정하면 이런 오류가 발생한다.

The executable ~/.pythonz/pythons/CPython-2.7.6/bin/python (from --python=~/.pythonz/pythons/CPython-2.7.6/bin/python) does not exist

Python 3.3 이상에서는 다음과 같이 하면 된다. pythonz로 설치한 python 파일이 있는 경로까지 들어가서 pyenv 명령을 실행한다. pyenv 명령에 대해서는 https://docs.python.org/3/library/venv.html를 참고한다.

cd .pythonz/pythons/CPython-3.4.3/bin
pyvenv ~/myvenv

Python 2.X는 내 알 바가 아니므로 여기에는 요약조차 하지 않는다.
