참고: http://stackoverflow.com/questions/19856927/install-sip-pyqt-on-virtualenv

  1. virtualenv를 활성화한다(ex: source virtualenv_prj/bin/activate).
  2. cd virtualenv_prj
  3. pip install --no-install sip를 실행한다. 마지막에 에러가 뜨지만 무시한다.
  4. cd build/sip
  5. python configure.py && make && make install
  6. cd ../.. (virtualenv_prj 폴더)
  7. PyQt홈페이지에서 Source Packages 중에 자신의 플랫폼에 해당하는 파일을 다운로드한다.
  8. 압축을 풀고 그 폴더(PyQt-<flatform>-gpl-<version>)로 들어간다.
  9. python configure.py && make && make install  실행

끝.
