요근래 Sublime Text 2 사용자가 많이 늘고 있다. 가볍고 빠르고 이쁘다. 기능도 많다.
게다가 상용이지만 (아직까지는) 기능 제한없이 계속 사용할 수 있다. 구입하지 않으면 파일을 저장할 때 구입안내창이 뜬다. 매번 뜨는 게 아니니 돈 안내고 쓰려면 이 정도는 감수하자.
얼마나 강력한지는 Sublime Text 홈페이지에 올라온 동영상만 봐도 알 수 있다.
http://www.sublimetext.com/blog/articles/sublime-text-2-0-released

Mac용  LaTeX 에디터로 주로 TeXShop이나 MacVim을 썼는데, Sublime Text 2를 사용해 보려고 설정을 했다.

  1. 준비물: MacTeX 2010 이상, Skim(Mac OS용 PDF 뷰어). 이 두 개의 프로그램을 다운받아서 설치한다.
  2. Sublime Text 2 를 다운받아서 설치한다.
  3. SublimeText / LaTeXTools 을 다운받는다. zip파일을 풀면 LaTeXTools-Master 따위의 이름으로 된 폴더가 나오는데, 적당히 'LaTeXTools' 정도로 바꾼다. 절대 'LaTeX'이라는 이름으로 바꾸면 안된다!
  4. Sublime Text 2를 실행시킨다. 메뉴에서 Preferences | Browse Packages 를 선택한다.
  5. 파인더가 실행되면서 Packages라는 폴더가 열릴 것이다. Sublime Text 2에서 지원하는 언어들이 각 폴더로 구분되어 있다. 엄청 많다. 이 중에 'LaTeX'폴더도 있다. 위 2번에서 'LaTeX'이라는 이름을 왜 쓰면 안되는지 이해할 수 있을 것이다.
  6. Packages 폴더에 'LaTeXTools'라는 폴더를 넣어준다.  이제 Sublime Text 2에서 할 일은 끝났다. Sublime Text 2를 재시작할 필요도 없다.

이제 inverse search를 하기 위해 Skim을 설정한다.

Skim을 실행해서 Preferences를 열면 Sync탭이 보인다. Sync탭에서 각 항목을 아래와 같이 설정한다.

  • Check for file changes: 체크하지 않는다.
  • Preset: Custom
  • Command: /Applications/Sublime text.app/Contents/SharedSupport/bin/subl
  • Arguments: "%file":%line
설정은 끝.

Sublime Text 2에서 LaTeX 문서를 작성한 후, 단축키 Cmd + B를 누르면 컴파일이 되고, 바로 Skim에서 PDF를 보여준다. 이 때, Sublime Text 2의 커서 위치에 해당하는 부분이 Skim에서 하이라이팅된다.

 

다른 단축키는 https://github.com/SublimeText/LaTeXTools를 참고.
