오늘 좋은 Vim 플러그인을 발견했다. 이름하야 NERDCommenter

https://github.com/scrooloose/nerdcommenter

각 언어에 맞게 주석처리를 쉽게 할 수 있게 해준다. vimrc에 설정하지 않아도 대부분의 기능을 쉽게 사용할 수 있다.

아주 유용한 플러그인이라서 배워서 쓸 겸 사용법을 한글로 옮겨봤다. 옮기지 않은 부분은 내가 이해하지 못했거나 어떻게 쓰는지 모르는 것들이다. 옮기면서 내 생각도 섞었다.

http://spf13.com/post/vim-plugins-nerd-commenter에서 구체적인 예를 볼 수도 있다.

1. 사용법

이 키 맵팅은 기본으로 제공된다(메뉴로도 제공됨).

아래의 대부분의 맵핑은 normal/visual 모드에서만 쓰인다. |NerdComInsertComment|맵핑은 insert 모드에서만 쓰인다.

[count]<leader>cc |NERDComComment|
현재 줄이나 visual모드에서 선택된 텍스트를 주석처리

[count]<leader>cn |NERDComNestedComment|
Same as <leader>cc but forces nesting.

[count]<leader>c |NERDComToggleComment|
Toggles the comment state of the selected line(s). If the topmost selected line is commented, all selected lines are uncommented and vice versa.

[count]<leader>cm |NERDComMinimalComment|
여러 줄을 주석처리할 때 주석기호를 한 번만 사용(말로 하니 이상한데, 여러 줄 선택한 상태에서 해보면 됨). 파일 타입에 따라 안되는 언어들도 있음(예: 파이썬)

[count]<leader>ci |NERDComInvertComment|
각 선택한 줄의 주석 상태 토글. 여러 줄을 선택한 상태에서도 쓸 수 있음

[count]<leader>cs |NERDComSexyComment|
선택한 줄을 '멋있게' 주석처리

[count]<leader>cy |NERDComYankComment|
첫번째 줄이 복사(yanked)된다는 것만 빼면 <leader>cc와 같다.

<leader>c$ |NERDComEOLComment|
현재 줄에서 커서 위치부터 줄 끝까지 주석처리

<leader>cA |NERDComAppendComment|
줄 끝에 주석 기호를 추가하고 입력 상태로 바꿈

|NERDComInsertComment|
Adds comment delimiters at the current cursor position and inserts between. Disabled by default.

<leader>ca |NERDComAltDelim|
주석 기호를 다른 것으로 바꿈(vimrc에 설정이 되어있어야 한다)

[count]<leader>cl
[count]<leader>cb |NERDComAlignedComment|
주석 기호를 정렬해주는 것만 제외하면 |NERDComComment|와 같다. cl은 왼쪽만 정렬해주고, cb는 양쪽 다 정렬해준다.

[count]<leader>cu |NERDComUncommentLine|
선택한 줄 주석 해제

2. 요약

거의 모든 기능이 [count]와 함께 쓸 수 있기 때문에 인라인 주석, 블록 주석이라는 구분은 애매하다. 다만 내 기준으로 보면 이런 식으로 쓰는 게 좋은 것 같다.

  • 블럭 주석처리: cb, cl, cm, cs
  • 인라인 주석처리: cc, cn,
  • 주석 처리 토글: ci
