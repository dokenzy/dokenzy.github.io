"2014-02-06"이라는 문자열을 구했다. 이 문자열로 Datetime객체를 만들고 싶다.

import datetime
reg_date = datetime.datetime.strptime("2014-02-06", "%Y-%m-%d").date()

끝.

만약 문자열이 "20140206" 이런 형식이라면

import datetime
reg_date = datetime.datetime.strptime("20140206", "%Y%m%d").date()

끝.
