pythonz: 파이썬 설치 관리자

개요

pythonz은 사용자의 $HOME에 파이썬을 자동으로 빌드하고 설치하는 프로그램이다. pythonbrew에서 fork했다.

pythonbrew 프로젝트가 중단된 것으로 보인다. 그리고 내가 정말로 원하지 않는 기능들도 있부 있었다. 그래서 내가 쓰기 좋게 좀 더 간단하게 만들었다.

CPython, Stackless, PyPy 그리고 Jython을 지원한다.

설치

pythonz를 다운로드하고 설치할 때는 이 방법을 추천한다. 쉘에서 이 문장을 실행한다.

curl -kL https://raw.github.com/saghul/pythonz/master/pythonz-install | bash

또는

fetch -o - https://raw.github.com/saghul/pythonz/master/pythonz-install | bash

그러면 pythonz가 ~/.pythonz에 설치된다.

~/.bashrc 맨 끝에 다음 줄을 추가한다(역주: 맥은 ~/.bash_profile)

[[ -s $HOME/.pythonz/etc/bashrc ]] && source $HOME/.pythonz/etc/bashrc

pythonz를 다른 곳에 설치하려면 PYTHONZ_ROOT 환경변수를 설정해야 한다.

export PYTHONZ_ROOT=/path/to/pythonz
curl -kLO https://raw.github.com/saghul/pythonz/master/pythonz-install
chmod +x pythonz-install
./pythonz-install

시스템 전체에 설치하기.

설치 스크립트를 root권한으로 실행하면 pythonz가 자동으로 /usr/local/pythonz 에 설치된다.

root권한으로 설치하면 시스템의 모든 사용자에게 pythonz가 자동으로 설정된다.

사용법

pythonz command [options] version

파이썬 설치하기:

pythonz install 2.7.3
pythonz install -t stackless 2.7.2
pythonz install -t jython 2.5.2
pythonz install -t pypy --url https://bitbucket.org/pypy/pypy/downloads/pypy-1.8-osx64.tar.bz2 1.8
pythonz install --verbose 2.7.2
pythonz install --configure="CC=gcc_4.1" 2.7.2
pythonz install --url http://www.python.org/ftp/python/2.7/Python-2.7.2.tgz 2.7.2
pythonz install --file /path/to/Python-2.7.2.tgz 2.7.2
pythonz install 2.7.3 3.2.3

설치한 파이썬 버전 목록:

pythonz list

설치할 수 있는 파이썬 버전 목록:

pythonz list -a

특정 파이썬 제거하기:

pythonz uninstall 2.7.3
pythonz uninstall -t stackless 3.2.2

오래된 소스폴더와 아카이브(archives) 삭제:

pythonz cleanup

pythonz를 최신 버전으로 업그레이드하기:

pythonz update

pythonz로 설치한 파이썬은 virtualenv같은 것을 통해서 사용하기를 추천한다.

mkvirtualenv -p ~/.pythonz/pythons/CPython-2.7.3/bin/python python2.7.3

virtualenv에 대한 자세한 설명은 virtualenv 문서 참고바람(역주: virtualenv+pythonz 같이 사용하기 참고)

명령어

사용할 수 있는 명령어 보기:

pythonz help

각 명령어에 대한 도움말 보기:

pythonz help <command>
