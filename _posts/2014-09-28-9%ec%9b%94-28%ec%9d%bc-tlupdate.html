animate: A LaTeX package for creating portable, JavaScript driven PDF animations from sets of vector graphics or raster image files or from inline graphics. 신기신기. 음성도 들어가는데 이것도 animate로 한 것인지는 잘 모르겠음. adobe reader에서 됨(Skim에서는 안됨)

  1. exsheets:  ExSheets provides means to create exercises or questions and their corresponding solutions.
  2. getmap: 간단한 샘플이라도 만들어 봐야겠다. 근데 우리나라도 되려나...?
  3. tcolorbox: tcolorbox provides an environment for colored and framed text boxes with a heading line.
  4. graphbox: graphbox is an extension for the LATEX2ε package graphicx to facilitate the placement of graphics relative to the current position using additional optional arguments [...] of \includegraphics.
  5. xq: The xq package is made for writing about Xiangqi or Chinese chess. 장기판 만들어주는 패키지. 신기하다. move 명령으로 말을 움직일 수 있다. 근데 왕 장기알이 다른 알과 크기가 같아서 좀 아쉽다. 언어를 선택하면 영어, 스페인어, 독일어, 체코어, 프랑스어 등으로 알을 선택할 수 있나본데, 어떻게 쓰는건지는 잘 모르겠다. sty파일에 한국어를 추가해볼까 한다. xq.sty 파일을 열어보니 한국어를 추가할 필요가 없겠다. 마, 상, 포 이런 식으로 이름을 붙일 수 있나 싶었는데 그런 거 없다. 그냥 알파벳 하나씩 대입하는 식인데, 한글은 대소문자도 없다보니 팀 구별도 못한다. 그냥 english로 써야겠다.
