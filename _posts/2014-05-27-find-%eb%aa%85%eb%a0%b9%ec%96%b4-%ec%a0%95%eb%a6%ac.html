정리 중...

  • find ./ -type f -name 'abc.pdf' 현재 디렉토리(하위 디렉토리 포함) 밑에서 abc.pdf 파일 찾기
  • find ./ -type f -iname 위와 동일한데, 대소문자 구분하지 않음.
  • find ./ -type d -name 'documents' 현재 디렉토리에서 documents 디렉토리 찾기
  • find ./ -type d -iname 'documents' 위와 동일한데, 대소문자 구분하지 않음.
  • find ./ -type f -name '*.html' | xargs grep 'angular.js' 현재 디렉토리에서 angular.js라는 텍스트가 있는 html 파일 찾기
