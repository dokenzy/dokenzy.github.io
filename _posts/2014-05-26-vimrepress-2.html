VimRepress

  1. 오리지널 소스는 http://www.vim.org/scripts/script.php?script_id=3510 에서 받음.
  2. 블로그 글 보기, 새 글 작성하기, 글 수정하기 기능 등이 제공됨.
  3. 워드프레스 3.5부터는 XML-RPC로 글쓰기 하는 기능이 기본제공되므로, 워드프레스에서 뭔가 설정할 필요는 없다.
  4. 마크다운이나 HTML로 글을 쓸 수 있다.
  5. 카테고리 자동으로 불러오는 단축키가 있다는데, 내 맥에서는 동작하지 않았다.
  6. BlogOpen 명령에서 오류 발생해서 약간 수정했음.
  7. 참고로 이 포스트도 VimRepress로 작성했다.

설치

필수조건

  1. vimrepress 내부에서 python을 사용하므로 python을 사용할 수 있는 Vim이 있어야 한다. 터미널에서 vim --version 이라고 입력하면 나오는 화면에서 +python이 보이면 된다. 관련글 참고 MacVim 직접 빌드 & 설치하기
  2. 파이썬용 markdown을 설치한다. pip install markdown 명령으로 쉽게 설치할 수 있다.

vimrepress 설치

  1. http://github.com/dokenzy/vimrepress 에서 파일을 다운로드받은 후, .vim 디렉토리에 복사한다.
  2. 홈디렉토리에 .vimpressrc 파일을 만든다.
  3. ~/.vimpressrc 파일을 다음과 같은 형식으로 편집한다. 
    [Blog0]
blog_url = http://blog.dokenzy.com
username = admin
password =

     

  4. password에 비밀번호를 넣으면 Vim으로 접근할 때 비밀번호를 묻지 않아서 편하지만, 플레인 텍스트로 저장되므로 절대 추천하지 않는다. 비밀번호를 저장해 두지 않더라도 저 항목은 두어야 한다.

명령어

  1. BlogList: 최근 포스트 30개
  2. BlogList page: 최근 페이지 30개
  3. BlogList post 100: 최근 포스트 100개
  4. BlogNew post: 새 포스트 쓰기
  5. BlogNew page: 새 페이지 쓰기
  6. BlogSave: 저장하고 공개하기
  7. BlogSave draft: 임시글 상태로 저장하기
  8. BlogPreview local: 웹브라우저에서 포스트나 페이지 미리보기
  9. BlogPreview publish: BlogSave publish 실행 후 브라우저로 여는 것과 같다.
