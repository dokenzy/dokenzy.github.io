bower가 참 편하긴한데, command not found 오류가 발생하는 경우가 종종 있다. 이건 bower 문제라기 보단 npm 문제같은데, 멀쩡히 되던 것도 안되는 경우가 종종 있다. meteor도 내부적으로 node를 이용하는데, meteor는 이런 문제는 없는 것 같다.

아무튼 이런 오류가 발생할 때 해결방법

참고: http://stackoverflow.com/questions/12369390/bower-command-not-found

$ npm config set prefix /usr/local
$ npm install -g bower

$ which bower
>> /usr/local/bin/bower

npm install 명령은 root 권한으로 실행한다.
