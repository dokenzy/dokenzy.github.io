모 커뮤니티에 올라온 질문.

Vim에서 HTML파일을 편집하는데, <script></script>안에서 자바스크립트 코드가 자동 들여쓰기가 안된다고.

vim-javascript는 일단 내 Macvim에서 원하는 기능이 되지 않았다.

어쩔 수 없이 Vim에서 set 명령으로 파일타입을 변경해야 하나?

:set filetype=javascript(또는 js)

이 방법은 확실하긴 하지만 번거롭다.

다행히 좋은 플러그인을 발견했다. 이름하야 html improved indentation: A better indentation for HTML and embedded javascript

~/.vim 폴더 밑에 indent 폴더를 만들고, 거기에 다운받은 html.vim 파일을 넣으면 된다.

끝.
