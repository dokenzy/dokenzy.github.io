스택오버플로우에서 답을 찾았다.(링크)

답을 인용하면:

I'm assuming you are on python 2 - then use assertRegexpMatches instead of assertRegex.

assertRegex was introduced in python 3:
Changed in version 3.2: The method assertRegexpMatches() has been renamed to assertRegex().

이 책에서는 Python3를 기준으로 하고 있는데, 나는 Python 2.7.5에서 작업하고 있다. Python 2.7.5에서는 assertRegex()함수의 이름은 assertRegexpMatchs()이다.
