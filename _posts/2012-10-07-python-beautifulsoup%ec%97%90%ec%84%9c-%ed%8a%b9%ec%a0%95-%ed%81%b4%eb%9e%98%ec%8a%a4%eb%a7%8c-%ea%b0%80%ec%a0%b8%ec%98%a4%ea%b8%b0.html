파이썬에는 BeautifulSoup라는 걸출한 HTML 파서가 있다. BeautifulSoup는 태그와 id, class 등을 이용하여 파싱을 쉽게 할 수 있다.

오늘 간단한 유틸리티를 만드는 데 메모해 둘만한 사항이 있어 남겨둔다.

from bs4 import BeautifulSoup
#BeautifulSoup 버전 4부터는 모듈 이름이 bs4로 바뀌었다.
soup = BeautifulSoup('myfile.html')
print soup.find_all("ul", class_="sister"))

bs4 문서에는 저렇게 하면 클래스가 sister인 ul요소를 가져온다고 하는데, 직접 해보니 동작이 안된다. 오류는 없지만 빈 리스트만 리턴한다.

구글링 한 결과 다음과 같이 하면 원하는 기능을 수행할 수 있다.

print soup.find_all("ul", {"class", "sister"})

위와 같이 클래스 정보를 세트(set) 형식으로 주면 클래스가 sister인 ul요소를 가져올 수 있다. 만약 'sister'외에 'brother' 클래스도 같이 가져오고 싶다면,

print soup.find_all("ul", {"class", "sister", "brother"})

와 같이 하면 된다.

공식 문서가 잘못됐을 경우보다는 내가 실수했을 확률이 더 높겠지만, 어쨌든 이런 식으로 일단 해결.
