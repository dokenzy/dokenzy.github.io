제 블로그에서 사용하는 코드 하이라이팅 플러그인은 Crayon Syntax Highlighter입니다.

유명한 플러그인인데 설정화면 언어에 한글이 없네요. 저보다 영어 잘하고 워드프레스 잘 하는 사람이 널렸는데 왜 번역이 안되어 있을까요? 음... 그런 분들은 영어가 불편하지 않은가봅니다.

결국 제가 답답해서 토익 500점도 안되는 실력으로 번역했습니다. 더 좋은 번역을 제안해 주시면 고맙겠습니다.

개발자인 aramk씨에게 pull request 보냈는데 하루만에 머지해 주네요. 조금 전 업데이트한 파일도 pull request 보냈으니 머지않아 머지해 줄 것 같습니다. aramk씨가 플러그인을 업데이트하기 전까지는 저장소에서 받으시면 됩니다.

aramk씨의 저장소는 crayon-syntax-highlighter@Github입니다.

필요하신 분 가져다 쓰시고 영어 잘하시는 분들 번역 좀... ㅜㅜ
