코드: https://github.com/dokenzy/unichar

예전에 회사일 때문에 급하게 만들었던 프로그램. Mac OS X와 윈도우즈에서 돌아간다(윈도우즈에서 만들었는데,  Mac에서 실행해 봤더니 잘 된다. 파이썬 만쉐~~~)

바뀐점

  • v0.7: Python 버전을 3.3으로 바꿈. 고대 이집트 상형문자같은 것도 처리할 수 있음.
  • v0.6: 10진수 추가, 카테고리 정보 추가, 유니코드 버전 추가, 글자 큼지막하게 보여주는 기능 추가

사용법

  1. Character 칸에 문자를 입력하고 엔터키를 치면 아래에 해당 유니코드 코드포인트와 문자의 이름이 보인다.
  2. '가'같은 글자는 엔터키를 두 번 쳐야된다.
  3. more info를 누르면 http://www.fileformat.info 사이트로 이동하여 더 자세한 정보를 보여준다.
  4. Unicode Hex 에는 유니코드 코드포인트나 16진수 값을  'AC00', 'U+AC00' 형식으로 입력한다. 대소문자는 가리지 않는다.
  5. Unicode Dec에는 유니코드 코드 포인트의 10진수 값을 입력한다.
