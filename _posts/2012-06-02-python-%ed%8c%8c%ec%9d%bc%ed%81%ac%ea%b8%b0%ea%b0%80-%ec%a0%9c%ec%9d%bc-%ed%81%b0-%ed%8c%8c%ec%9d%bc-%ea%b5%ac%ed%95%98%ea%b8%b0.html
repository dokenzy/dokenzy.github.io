여기 올라온 답변

import os, glob
largest = sorted( (os.path.getsize(s), s) for s in glob.glob('yourdir/*.avi') )[-1][1]

위 코드와 동일하지만 쉽게 풀어쓴 코드

import os, glob
files = glob.glob('yourdir/*.avi')
largest = sorted((os.path.getsize(s), s) for s in files)[-1][1]

sorted() 함수 설명: http://docs.python.org/2/library/functions.html#sorted
