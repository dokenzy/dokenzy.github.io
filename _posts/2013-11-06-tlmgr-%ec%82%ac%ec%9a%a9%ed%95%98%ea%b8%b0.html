텍라이브 매니저인 tlmgr 대충 사용법

tlmgr update --self --all : 패키지 업데이트. CTAN의 미러 중에 자동으로 선택된 미러사이트로부터 업데이트한다.

tlmgr repository : 내 텍 시스템에 등록된 텍라이브 저장소 리스트(tlmgr repository list)와 같다. 이 때 반드시 main이라는 이름을 갖는 저장소가 있어야 한다.

tlmgr repository add <저장소 주소> <별명>
ex)

tlmgr repository add http://ftp.ktug.org/KTUG/texlive/2013 kotex

 

tlmgr repository remove <저장소 주소 또는 별명>
ex)

tlmgr repository remove kotex

맨 처음에 소개된 명령을 조금 바꾸어 보자.

sudo tlmgr --repository=ctan update --self --all

위와 같이 업데이트 저장소를 지정해 줄 수 있다.
