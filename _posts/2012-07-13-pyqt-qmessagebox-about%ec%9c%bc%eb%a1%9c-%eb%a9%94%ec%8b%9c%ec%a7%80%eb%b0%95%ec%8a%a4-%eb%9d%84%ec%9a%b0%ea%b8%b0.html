QMessageBox로 간단한 정보를 제공하는 창을 띄우는 방법

__version__ = "1.0"
QMessageBox.about(self, "About This", """<b>My Utility</b> v{0}""".format(__version__))

about 외에 warning, critical, information 등이 있다. 각각 아이콘이 다르므로 용도에 맞게 사용하면 된다.

메시지 박스에 OK, Cancel 같은 버튼도 추가할 수 있다.

ans = QMessageBox.warning(self, "Oops", "there is a error.", QMessageBox.Ok|QMessageBox.Cancel)
if (ans == QMessageBox.Ok):
	return self.Foo()
else:
	raise Exception, sys.exit(1)

이렇게 하면 메시지박스에 OK와 Cancel 버튼이 생기고, 사용자가 누른 값이 ans 변수에 저장된다.

사용자가 OK를 눌렀는지 Cancel을 눌렀는지 확인하는 방법은 if (ans == QMessageBox.Ok) 와 같이 간단하게 할 수 있다.
