홈디렉토리에 .pypirc 파일을 만들고 다음과 같이 입력한다.

[distutils]
index-servers =
    pypi
    test

[pypi]
repository = https://pypi.python.org/pypi
username:dokenzy
password = 

[test]
repository = https://testpypi.python.org/pypi
username = dokenzy
password =

이제 pypi에 register하거나 upload하기 전에 아래 명령으로 테스트를 해볼 수 있다.

$ python setup.py register -r test

실제 pypi에 하려면 repository 이름을 pypi로 지정하면 된다.

$ python setup.py register -r pypi

username과 password 둘 다 입력하거나 둘 다 비워둔다. 입력하지 않으면 명령 실행시 인증을 해야한다.
