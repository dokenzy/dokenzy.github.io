    2. https://github.com/gmarik/Vundle.vim
  2. Pathogen
    1. manage your runtimepath
    2. https://github.com/tpope/vim-pathogen

  6. nerdcommenter

    2. https://github.com/scrooloose/nerdcommenter
  7. vim-multiple-cursors

  8. vim-markdown-preview

  9. vim-easymotion
    1. Vim motions on speed!

  10. neocomplcache
    1. Ultimate auto-completion system for Vim
    2. https://github.com/Shougo/neocomplcache.vim

  13. vim-surround
    1. surround.vim: quoting/parenthesizing made simple

  14. vim-airline

관심있는 플러그인

  1. wildfire
    1. Smart selection of the closest text object
    2. https://github.com/Shougo/wildfire.vim
  2. pytest
    1. Runs your UnitTests with py.test displaying red/green bars and errors
    2. https://github.com/alfredodeza/pytest.vim
  3. vim-autoclose
    1. This plugin for Vim enable an auto-close chars feature for you. For instance if you type an '(', ``autoclose`` will automatically insert a ')' and put the cursor between than.
    2. https://github.com/Townk/vim-autoclose
  4. powerline
    1. Powerline is a statusline plugin for vim, and provides statuslines and prompts for several other applications, including zsh, bash, tmux, IPython, Awesome and Qtile.
    2. https://github.com/Lokaltog/powerline
  5. rainbow_parentheses.vim
    1. Better Rainbow Parentheses
    2. https://github.com/kien/rainbow_parentheses.vim
  6. Gundo
    1. Graph your Vim undo tree in style.
    2. http://sjl.bitbucket.org/gundo.vim/
  7. vim-indent-guides
    1. A Vim plugin for visually displaying indent levels in code
    2. https://github.com/nathanaelkane/vim-indent-guides
