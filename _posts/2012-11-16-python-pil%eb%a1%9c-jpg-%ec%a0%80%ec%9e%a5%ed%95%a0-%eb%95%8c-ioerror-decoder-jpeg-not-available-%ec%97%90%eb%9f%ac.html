Mac에서 Python으로 PIL(Python Imaging Library)을 이용해서 이미지 리사이징 스크립트를 만들고 있다.

리사이징을 하고 저장을 하는데 에러가 생긴다.

IOError: decoder jpeg not available

뭥미?

다음과 같이 한다.

먼저 PIL이 깔려있다면 삭제부터 해주고, libjpeg을 설치한 다음, PIL을 다시 설치한다.

sudo pip uninstall PIL
brew install libjpeg
sudo pip install PIL

맥 OS를 매버릭스로 올리고 나니, brew install libjpeg  실행하니 /usr/local/lib에 link 할 수 없다는 오류가 발생한다. 이럴 때는 sudo chmod myid /usr/local/lib  명령을 먼저 실행한 후,

brew uninstall libjpeg
brew install libjpeg
sudo pip install pil

을 실행한다.

끝.
