파이썬으로 현재 시스템의 timezone을 구하는 방법을 찾아봤는데, 다 번거롭다. 한참 검색한 끝에 tzlocal(https://github.com/regebro/tzlocal)이라는 라이브러리를 찾았다. 아주 맘에 든다.

from tzlocal import get_localzone
tz = get_localzone()
# 결과: Asia/Seoul

그 외 이런 방법도 있다. 출처(Get time zone information of the system in Python?)

import time
offset = time.timezone if (time.localtime().tm_isdst == 0) else time.altzone
tz_offset = offset / 60 / 60 * -1
# 결과: 9.0

경우에 따라 이것이 더 필요한 경우도 있을 수 있겠다.
