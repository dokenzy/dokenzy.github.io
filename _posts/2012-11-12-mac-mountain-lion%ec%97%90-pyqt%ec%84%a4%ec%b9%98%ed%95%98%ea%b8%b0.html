Mac의 OS X에는 Python이 기본으로 설치되어 있다. PyQt만 깔면 된다.

하지만 문제가 있다. PyQt를 쉽게 설치할 수 있는 맥용 바이너리 설치파일인 PyQtX가 Python 2.7.3이 없다며 설치가 안된다.

일단 맥의 기본 Python 버전부터 업그레이드하자. Mountain Lion에 설치된 파이썬 버전은 2.7.2이다.

Python 공식 홈페이지에서 최신 버전(현재 2.7.3)을 받아서 설치한다. 설치가 끝나면 응용 프로그램 폴더에 Python 2.7 이라는 폴더가 생긴다. 이 폴더에 보면 'Update Shell Profile.command'이라는 파일이 있다. 이걸 실행한다.

터미널을 실행한다. 터미널이 이미 실행중이었다면 종료하고 다시 실행한다.

파이썬 버전을 확인한다.

python --version

2.7.3이다.

이제 아까받은 PyQtX설치 파일을 다시 실행한다. 설치가 끝나면 터미널에서 PyQt4를 임포트해본다.

import PyQt4

 

끝.
