node.js 설치

node.js용 웹 어플리케이션 프레임웍. 따라서 node.js를 먼저 설치한다. http://nodejs.org/download/에서 다운로드하거나 brew install node.js 명령으로 설치할 수 있다.

node.js를 설치하면 npm이라는 node package manager가 설치된다. npm을 이용하여 express를 설치한다. 이 때 -g 옵션을 주면 시스템의 어떤 경로에서든 express를 사용할 수 있다.

한국어 문서 http://firejune.io/express/

angular.js

angular.js는 구글에서 만든 프론트엔드 자바스크립트 프레임워크. bower로 설치하는 것이 좋다.

bower install angular
